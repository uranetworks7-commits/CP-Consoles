# **App Name**: GameNexus

## Core Features:

- Game Library Display: Showcase a collection of available web-based games, including a title and entry point for each game.
- Game Launch Pad: Allow users to select a game from the library and launch it directly within the application interface, such as via an iframe or new tab.
- External Game Integration: Configure and integrate specified external game URLs, like 'https://2212022025.github.io/Car-Game/' and 'https://ura-battle.netlify.app/', making them accessible within the console.
- Responsive Game Display: Ensure games are displayed in a responsive manner, adapting to various screen sizes for an optimal user experience.

## Style Guidelines:

- Primary accent color: Vibrant blue-violet (#336BE0) to evoke modern technology and energy. Background color: Dark desaturated blue-gray (#1F262E) for an immersive gaming environment. Accent color: Bright cyan (#44CAE6) for interactive elements and highlights, creating good contrast.
- Body and headline font: 'Space Grotesk' (sans-serif) for a modern, techy, and clean aesthetic, suitable for digital interfaces.
- Use sharp, modern icons with a minimalist style, possibly incorporating gaming controller motifs or abstract geometric shapes.
- A grid-based layout for the game library to allow for easy browsing, transitioning to a full-screen view with minimal interface elements when a game is active.
- Subtle, fast, and fluid transition animations for game loading and navigation between screens, providing a polished and responsive user experience.
